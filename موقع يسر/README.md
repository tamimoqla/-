يسر — موقع تعليمي عربي

كيفية العرض محليًا:

1. افتح المجلد `c:/Users/Tamim/Desktop/My website` في مستكشف الملفات.
2. انقر نقرًا مزدوجًا على `index.html` لفتحه في المتصفح.

المميزات المتضمنة:
- رأس الصفحة مع الروابط السريعة وقائمة الحساب على الجانب الأيمن.
- مربع بحث يبحث عن الصفوف والمواد والوحدات والدروس أو أسماء المعلمين.
- قسم الدورات منظم حسب الصفوف والمواد مع شروحات.
- قسم الصور (صور عينة).
- قسم المعلمين مع بطاقات المعلم الأربعة.
- آراء العملاء والشهادات (دوارة).
- نموذج صندوق الاقتراحات والشكاوى والدعم (يتم الحفظ في `localStorage`).
- تمرير سلس وأزرار يمكن الوصول إليها وتخطيط سريع الاستجابة.
- اتجاه RTL كامل للعربية.

الخطوات التالية (اختياري):
- ربط النماذج بخادم لتخزين الرسائل على الخادم.
- إضافة المزيد من الصفوف والمواد واستيراد المحتوى من ملف JSON أو CMS.
- إضافة المصادقة لقائمة الحساب وصفحات الملف الشخصي.
